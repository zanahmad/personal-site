Neural Operators for PDEs

In scientific machine learning, data-driven deep learning methods aim to predict the solutions of partial differential equations (PDEs), avoiding the computationally expensive numerical integration required for large-scale simulations. This is especially beneficial for applications like domain optimization and precision medicine, where multiple PDEs need to be solved across varying parameters or domains, demanding significant computational resources.

Neural operators learn mappings between high-dimensional function spaces, allowing generalization across a family of PDEs without retraining for different parameters or conditions. Mathematically, a neural operator ( \mathcal{N}{\theta} ) is defined as a mapping from an input domain function space ( \mathcal{A}(\Omega{\alpha};\mathbb{R}^{d_a}) ) to an output target function space ( \mathcal{U}(\Omega_{\alpha};\mathbb{R}^{d_u}) ):

[ \mathcal{N}{\theta}: \mathcal{A}(\Omega{\alpha};\mathbb{R}^{d_a}) \rightarrow \mathcal{U}(\Omega_{\alpha};\mathbb{R}^{d_u}), ]

where ( \theta \in \Theta ) denotes the neural operator’s parameters, and ( \Omega_{\alpha} \subset \mathbb{R}^d ) (or a ( d )-dimensional manifold) represents the spatial domain on which the functions are defined. The parameter ( \alpha \in \mathscr{A} ) denotes the shape of the domain. The dimensions ( d_a ) and ( d_u ) represent the sizes of the input and output function spaces, often as subspaces of Sobolev spaces.

Neural operators can be applied to problems described by continuous functions ( C(\Omega;\mathbb{R}^{d_a}) ) or Sobolev spaces ( H^s(\Omega;\mathbb{R}^{d_a}) ) for some ( s \geq 0 ). The neural operator ( \mathcal{N}{\theta} ) approximates the true target operator ( \mathcal{N} ) (e.g., the solution operator of a PDE), learned by training on input-output function pairs ( {a_i, u_i}{i=1}^{m} ), where ( a_i \in \mathcal{A} ) and ( u_i = \mathcal{N}(a_i) \in \mathcal{U} ).

For example, in cardiac electrophysiology, the input space ( \mathcal{A} ) might represent the initial electrical activation patterns across cardiac tissue, and the output space ( \mathcal{U} ) could correspond to the resulting electrical potential fields over time.
